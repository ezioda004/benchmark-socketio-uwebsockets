This repo contains benchmark module for socketio and uwebsockets.

## Requirements
- Node 20.3
- pm2

## Installation

### Server
```
cd benchmark-socketio-uwebsockets
npm i -g pm2
npm ci
npm run build
npm run start:server
```

### Client
```
cd benchmark-socketio-uwebsockets
npm i -g pm2
npm ci
npm run build
npm run start:client
```
